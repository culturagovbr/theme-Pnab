# entity-location (cópia do tema Pnab)

Cópia do componente do módulo **Entities** (`src/modules/Entities/components/entity-location/`) para permitir edição no tema Pnab.

## Alterações nesta cópia

- Inclusão de **mensagens de erro de validação** para os campos Estado e Município quando são selects ou inputs de texto (uso de `entity.__validationErrors['En_Estado']` e `entity.__validationErrors['En_Municipio']`), alinhado ao comportamento dos outros campos que usam `entity-field`.
